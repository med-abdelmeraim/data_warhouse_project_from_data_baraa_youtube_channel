## 📖 Project Overview

This project is a PostgreSQL implementation of the **Data Warehouse Project** originally created by the **Data with Baraa** YouTube channel. While the original project is built using **Microsoft SQL Server**, I recreated the complete solution in **PostgreSQL**, adapting the SQL syntax, ETL processes, and database-specific features.

The project follows the **Medallion Architecture** (Bronze, Silver, and Gold) to build a modern data warehouse from raw CSV files to business-ready analytical data.

### Key Features

* Designed a modern data warehouse using the **Bronze–Silver–Gold** architecture.
* Built ETL pipelines to extract, clean, transform, and load data into PostgreSQL.
* Developed fact and dimension tables using a **star schema** for analytical queries.
* Performed data cleansing, standardization, and integration of ERP and CRM datasets.
* Created SQL-based analytical queries to generate insights into customer behavior, product performance, and sales trends.
* Documented the data architecture, data model, ETL workflow, and project structure.
